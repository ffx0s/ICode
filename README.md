## 项目版本

`"react": "16.0.0-alpha.12"`  
`"react-native": "^0.48.4"`  
`"eslint": "^4.3.0"`

## 运行

1. `git clone https://github.com/ffx0s/ICode.git`
2. `cd ICode` && `yarn install`
3. yarn start

## 目录结构

```
.
├── src            # 项目文件目录
│   ├── api        # 接口文件
│   ├── components # 通用组件
│   ├── data       # JSON数据
│   ├── util       # 工具方法
│   ├── views      # 页面视图
```
