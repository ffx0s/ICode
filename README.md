## 项目版本
`"react": "16.0.0"`  
`"react-native": "^0.50.3"`  
`"eslint": "^4.3.0"`

## 功能
- [x] 知乎日报主题列表、主题详情 
- [x] V2ex主题列表、主题详情 
- [x] Github trending列表、仓库详情 
- [x] APP主题更换

## 运行

1. `git clone https://github.com/ffx0s/ICode.git`
2. `cd ICode` && `yarn install`
3. react-native run-ios

## 目录结构

```
.
├── src               # 项目文件目录
│   ├── api           # 接口文件
│   ├── assets        # 资源
│   ├── components    # 通用组件
│   ├── data          # JSON数据
│   ├── util          # 工具方法
│   ├── routers       # 路由
│   ├── views         # 视图
```
